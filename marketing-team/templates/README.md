# Strategy Templates — Elastic Technologies Marketing Team

This folder contains all official strategy templates used by the content-strategist agent (Zoe) and the wider marketing team. Every strategy document, brief, or plan produced must reference and follow the appropriate template from this folder.

---

## Template Index

| Template File | Use For |
|---|---|
| [`content-brief.md`](content-brief.md) | Individual content pieces: blog posts, social posts, videos, newsletters |
| [`campaign-plan.md`](campaign-plan.md) | Multi-channel marketing campaigns (4+ weeks, cross-platform) |
| [`topic-cluster.md`](topic-cluster.md) | Pillar + cluster content architecture for SEO and authority building |
| [`content-calendar.md`](content-calendar.md) | Weekly or monthly social media and content posting schedules |
| [`case-study.md`](case-study.md) | Client project case studies for blog, social, and proposals |
| [`monthly-report.md`](monthly-report.md) | Monthly marketing performance reports and KPI summaries |
| [`graphics-brief.md`](graphics-brief.md) | Briefs for the graphics design team (social assets, banners, etc.) |
| [`market-research.md`](market-research.md) | Competitor analysis, trend research, and audience insight reports |

---

## How to Use These Templates

1. **Identify** the type of output you need to produce.
2. **Open** the corresponding template file.
3. **Copy** the full template into the relevant workspace folder (e.g., `04-Campaigns/active/`, `02-Content/blog-posts/`).
4. **Rename** the file to reflect the specific project (e.g., `2026-Q2-software-dev-campaign.md`).
5. **Complete** all sections, replacing placeholders in `[BRACKETS]` with real content.
6. **State the template used** at the top of every completed document.
