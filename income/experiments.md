## How to Run an Experiment

1. Name it clearly
2. Write the hypothesis ("If I do X, then Y will happen")
3. Set a deadline (max 2 weeks)
4. Track the result
5. Kill it if no signal after the deadline — move to next

**Rule: Max 2 active experiments at once. More = scattered. Less = stalled.**
