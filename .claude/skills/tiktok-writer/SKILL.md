# TikTok Script Writer

Write a TikTok video script for Simon Kazooba.

**Topic / Input:** $ARGUMENTS

## Creator Profile
- **Name:** Simon Kazooba
- **Niche:** AI, tech, software, entrepreneurship
- **Series:** AI Challenge (documenting AI tools/experiments)
- **Tone:** Casual, engaging, relatable — no corporate speak
- **Default length:** 60 seconds

## Step 1 — Collect Input

If $ARGUMENTS is empty or incomplete, ask Simon for:
1. **Topic** — What is the video about?
2. **Series** — AI Challenge, general tech, business, or personal?
3. **Key point** — The one thing viewers should learn or feel
4. **Length** — 30 sec, 60 sec, or 90 sec?
5. **Hook idea** — Does Simon already have an opening line? (optional)

## Step 2 — Write the Script

Use the length-appropriate structure from [examples/script-formats.md](examples/script-formats.md).
Use proven hook formulas from [examples/hook-formulas.md](examples/hook-formulas.md).

Key rules:
- First sentence must stop the scroll — no warm-up, no intro
- Short sentences. One idea per sentence.
- Write how Simon talks — casual, first person
- Use numbers and specifics ("saved 3 hours" not "saved lots of time")
- End with a reason to comment or follow

**AI Challenge framing:**
- Frame as a personal experiment: "I tried...", "I challenged myself to..."
- Show process or before/after
- Be honest — what worked and what didn't drives engagement
- Connect to real business use (Simon runs Elastic Technologies)

## Step 3 — Deliver Output

Provide all of the following:

1. **Script** — with `[VISUAL CUE: what to show on screen]` notes inline
2. **Hook alternatives** — 2-3 different opening lines to choose from
3. **Caption** — 1-2 sentences for the post
4. **Hashtags** — 5-8 relevant tags

## Step 4 — Save the Script

Save the completed script to:
`teams/marketing/03-Social-Media/post-drafts/[YYYY-MM-DD]-[topic-slug].md`
