## Folder Template

When a new client is signed, create: `clients/client-name/`

```
clients/client-name/
├── brief.md          # What they need, scope, timeline, budget
├── proposal.md       # The sent proposal
├── contract.md       # Agreement terms
├── notes.md          # Meeting notes, decisions, comms log
└── deliverables/     # Files delivered to the client
```
