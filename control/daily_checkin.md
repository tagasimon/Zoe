# Daily Check-In
_Run this before touching anything else. 5 minutes._

---

## 5 Questions — Answer Now

1. **Did I collect the SchoolConnect 1M UGX yet?**
   → Yes / No. If No — that is Task 1 today. No exceptions.

2. **What is Task 1 today?**
   → Write it here before opening any other file.

3. **Is there a proposal or outreach message I should send today?**
   → Yes / No. If Yes — it goes in today.md as Task 2 or 3.

4. **What am I most likely to procrastinate on?**
   → Name it. That is probably the thing that needs to happen first.

5. **What is the one thing that, if done today, makes the day a win?**
   → Write it. Make sure it is in today.md.

---

## Anti-Procrastination Rules

- No social media before Task 1 is complete.
- No "research" without a deliverable at the end of it.
- No more than 3 tasks per day. Adding a 4th means one gets dropped.
- If you are reading, planning, or thinking for more than 20 minutes — stop. Write something or code something.

---

## End-of-Day Review (2 minutes)

- Task 1: Done / Not Done. Why not?
- Task 2: Done / Not Done. Why not?
- Task 3: Done / Not Done. Why not?
- What distracted me today?
- Tomorrow's Task 1: _______________

_If Task 1 is not done: write down exactly what stopped you before closing the laptop._
